02. 요소 노드의 텍스트 조작

nodeValue

지금까지 살펴본 노드 탐색, 노드 정보 프로퍼티는 모두 읽기 전용 접근자 프로퍼티이다. Node.prototype.nodeValue 프로퍼티는 setter와 getter가 모두 존재하는 접근자 프로퍼티이다. nodeValue 프로퍼티 참조 시 노드 객체의 값(텍스트 노드의 텍스트)을 반환한다. 텍스트 노드가 아닌 노드(문서 노드, 요소 노드)의 경우 null을 반환한다

textContent

Node.prototype.textContent 프로퍼티는 setter, getter 모두 존재하는 접근자 프로퍼티이다. 요소 노드의 텍스트와 모든 자손 노드의 텍스트를 취득하거나 변경한다. 요소 노드의 textContent를 참조하면 요소 노드의 콘텐츠 영역 내의 텍스트를 모두 반환하며 이 때 HTML 마크업은 무시 된다. 요소 노드의 textContent에 문자열을 할당하면 모든 자식 노드가 제거 되고 할당한 문자열이 텍스트로 추가 된다. 이 떄도 HTML 마크업은 무시 된다.
