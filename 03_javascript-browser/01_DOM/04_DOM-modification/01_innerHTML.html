01. innerHTML

Element.prototype.innerHTML 프로퍼티는 setter, getter가 모두 존재하는 접근자 프로퍼티로 요소 노드의 HTML 마크업을 취득하거나 변경한다.

태그 엘리먼트의 값을 읽거나, 변경할 떄 innerHTML 속성을 사용함.

innerHTML 프로퍼티를 사용한 DOM조작은 구현이 간단하고 직관적이라는 장점이 있지만, 사용자로부터 입력 받은 데이터를 그대로 innerHTML 프로퍼티에 할당하는 것은 XSS(Cross-site Scripting attack) 파싱 과정에서 그대로 실행 될 가능성이 있다.

innerHTML 프로퍼티에 HTML 마크업 문자열을 할당하는 경우 요소 노드의 모든 자식 노드를 제거하고 할당한 HTML 문자열을 파싱하여 DOM을 변경한다는 단점도 있다. 또한 새로운 요소를 삽입 할때 삽입할 위치를 지정할 수 없다는 단점도 있다.
