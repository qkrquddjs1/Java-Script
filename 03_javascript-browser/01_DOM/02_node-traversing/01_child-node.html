01. 자식 노드 탐색

노드 탐색

요소 노드 취득 후 취득 요소 기점으로 DOM트리를 옮겨 다니며 부모, 형제, 자식 노드 등을 탐색해야 하는 경우가 있다. DOM 트리 상의 노드를 탐색할 수 있도록 Node, Element 인터페이스는 트리 탐색 프로퍼티를 제공한다.

공백 텍스트 노드

HTML 요소 사이의 스페이스, 탭, 개행 등의 공백 문자는 텍스트 노드를 생성한다.

자식 노드 탐색

  1. Node.prototpe.childNodes : 자식노드를 탐색하여 NodeList에 담아서 반환
  2. Node.prototpe.firstChile : 첫 번쨰 자식 노드를 반환
  3. Node.prototpe.lastChile : 마지막 번째 자식 노드를 반환
  1. Element.prototpe.children : 자식노드 중 요소 노드를 탐색하여 HTMLCollection에 담아서 반환
  2. Element.prototpe.firstElementChile : 첫 번쨰 자식 요소 노드를 반환
  3. Element.prototpe.lastElementChile : 마지막 번째 자식 요소 노드를 반환

자식 노드 존재 확인

Node.prototype.hasChildNodes 메서드를 사용하여 자식 노드 존재 여부를 확인한다. 텍스트 노드를 포함하여 자식 노드의 존재를 확인한다.
